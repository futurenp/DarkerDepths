
Darker Depths
=======

Darker Depths is an underground biome expansion mod.  
It currently adds 3 biomes:
* Sandy Catacombs
* Molten Cavern
* Glowshroom Forest

## Dependencies
* **Required** Geckolib > 4.8
* **Required** World Gen API (one of the below)
    * Biolith _(recommended)_
    * Terrablender

## Config

Biomes
* Biomes can be enabled / disabled.
* The climate parameters for each biome can be adjusted to depict where biomes can generate.

Shaders
* Darker depths uses custom shaders that are incompatible with (most if not all) shaderpacks. These can be disabled to fall back to 'vanilla counterparts' in the config.

Fog  
* Darker Depths implements a fog system that is intended to only interact with its own biomes and fog effects. This can be disabled and biome fog strength can be configured.
* Fog is also very commonly broken by other mods, so even effects that darken how much you can see (like paranoia) may not work.
